# new-school-sdk-sjtu

## Test Environment

Python 3.12.11

## Overview

This repo builds on [FarmerChillax/new-school-sdk](https://github.com/FarmerChillax/new-school-sdk), extending and refining it specifically for the Zhengfang Academic Affairs Management System deployed at **Shanghai Jiao Tong University**. It also fixes several bugs present in the original codebase.

For usage instructions, refer to the [original project's documentation](https://farmerchillax.github.io/new-school-sdk/). As an early-stage fork, this repo may not fully document every change made.
